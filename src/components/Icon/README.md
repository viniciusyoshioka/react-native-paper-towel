# Icon

Its an extension of `@react-native-vector-icons/material-design-icons`.

## Usage

```tsx
import { Icon } from "react-native-paper-towel"


export function App() {
    return (
        <Icon
            name={"settings"}
            size={32}
        />
    )
}
```

## Docs

### IconProps

Type of Icon props. It extends the `IconProps` from
`@react-native-vector-icons/material-design-icons`.

### Icon component

The Icon component. Its props are defined by [`IconProps`](#iconprops).
